# 티스토리용 변환본

- `everydaypeony-tistory.html`: 티스토리 스킨/페이지에 붙여넣기 쉽도록 CSS와 히어로 이미지를 포함한 단일 HTML 파일입니다.
- Google Fonts는 외부 로드가 필요합니다. 티스토리 스킨에서 외부 폰트 로드를 막지 않는지 확인하세요.
- 티스토리 글쓰기 에디터보다 `스킨 편집 > HTML` 또는 별도 페이지 HTML 삽입 방식 사용을 권장합니다.
- 원본 배포용 파일은 루트의 `index.html`, `styles.css`, `assets/peony-hero.png`입니다.
